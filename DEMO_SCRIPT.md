## Recording Notes

**Best practices:**
- Use 1.25-1.5x zoom for clarity
- Click deliberately (2 frame pause between actions)
- Speak clearly and moderately paced (not too fast)
- Use system audio only (no background music unless added in post)
- Record full screen (1920x1080 minimum)
- If any lag: re-record that section

**Loom Settings:**
- Camera: Off (screen only)
- Microphone: Built-in or external (test beforehand)
- Share: Public link
- Format: MP4 (for sharing)
