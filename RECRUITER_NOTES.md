## Questions to Ask This Candidate

1. **"What would you do differently if you built this again?"**
   - Shows learning. Bad answer: "Nothing, it's perfect."

2. **"When did you realize you were wrong about something?"**
   - Shows humility. Bad answer: "I'm never wrong."

3. **"What did you not have time to build?"**
   - Shows prioritization. Bad answer: "Everything is done."

4. **"How would you handle this if a competitor launched?"**
   - Shows competitive thinking. Bad answer: "We'd sue them."

5. **"If this failed, what would you learn?"**
   - Shows growth mindset. Bad answer: "It can't fail."
